# Oracle SQL Queries

This repository contains Oracle SQL and PL/SQL queries related to:

* **Reporting** (Oracle APEX & Financials)
* **RM Net Sales** Analysis
* **Performance Tuning** & Optimization
* **Hierarchical Queries** (Manager-Subordinate logic)

## 🛠 Technologies
* **Database:** Oracle Database
* **Languages:** SQL, PL/SQL

---

## 📂 File Inventory (Queries Detail)

| File Name | Purpose / Description | Key Features |
| :--- | :--- | :--- |
| `rm_hierarchy_net_sales.sql` | RM aur unke neechay tamam subordinates ki sales nikalne ke liye. | Hierarchy, CONNECT BY, START WITH |
| `rm_net_sales_report_month_wise.sql` | Monthly sales ki reporting aur aggregations ke liye. | GROUP BY, Date Filtering |
| `rm_hierarchy_aum_allocation.sql` | RM Hierarchy ke mutabiq AUM data aur fund categories. | Hierarchical Joins, AUM Reporting |
| `rm_hierarchy_ntb_report.sql` | RM Hierarchy ke mutabiq New To Bank (NTB) customers ka data. | Status Filtering, NTB Reporting |

---

## 🚀 Query Features Used
* **Hierarchical:** CONNECT BY, LEVEL, START WITH
* **Analytical:** GROUP BY, Aggregations (SUM, COUNT)
* **Filtering:** Date Range Filtering, Status Exclusions
